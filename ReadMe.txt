##BewerbungsAufgabe
is a Webservice that accept a POST request with a data structure (in JSON format) and make some process on it and return the result as Json file

##Requirements:
- Node.js
- Express
- Body Parser
- Rest Client
- npm
- mocha
- chai
- Heroku CLS
- Git
- Heroku account
- cmd

##Usage
-local PC: node server.js
- Local Heroku: node server.js
